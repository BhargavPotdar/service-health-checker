# Service Health Checker

## Overview
A shell-based automation script to monitor the status of critical
services on Linux systems.

## Problem Statement
Service failures can go unnoticed without continuous monitoring,
leading to downtime and operational issues.

## Solution
This script checks service status and logs health information to help
detect failures early.

## Features
- Service status checks
- Logging support
- Extensible alerting logic

## Tech Stack
- Linux
- Shell scripting (Bash)

## How to Run
chmod +x service_health_checker.sh  
./service_health_checker.sh
